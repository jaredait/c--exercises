1.	Escribir un programa en lenguaje C en el cual se cree una clase “Nodo” que se va a utilizar para la 
implementación de la lista que va a almacenar datos numéricos, esta debe realizar las siguientes opciones, 
que se deben presentar con un menú (considerar que los datos deben ingresarse de manera ordenada / considerar 
que para cada paso se debe imprimir la lista actualizada):

*-* 1.1.	Añadir al inicio de la lista
*-* 1.2.	Añadir al final de la lista
1.3.	Añadir un elemento después de una referencia (en caso de no encontrarla se debe mostrar un mensaje de error)
*-* 1.4.	Eliminar elemento del final
*-* 1.5.	Eliminar toda la lista
1.6.	Encontrar la posición de la primera ocurrencia de un número dado por el usuario
1.7.	Quitar el elemento que el usuario desee
*-* 1.8.	Ver la lista
